# Run Modularity Clustering

Internal function to run modularity optimization.

## Usage

``` r
RunModularityClustering(
  SNN = matrix(),
  modularity = 1,
  resolution = 0.8,
  algorithm = 1,
  n.start = 10,
  n.iter = 10,
  random.seed = 0,
  print.output = TRUE,
  temp.file.location = NULL,
  edge.file.name = NULL
)
```
